# Crypto Volatility & Risk Analyzer

A data-driven web application that analyzes cryptocurrency volatility and classifies assets into risk categories.

## Features
- Live crypto price fetching
- Volatility calculation
- Risk classification
- CSV & PDF report generation
- Interactive dashboard

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

3. Open browser at:
   http://localhost:8501
